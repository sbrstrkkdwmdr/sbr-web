Helpful

    help            links to this page. Also can be used to get more info on a command.
                    /help command:(commandname)
                    options: 

    ping            Sends a ping to the bot and API.
                    /ping

    math            Calculates a simple math equation. use the text command if you want to do an equation with multiple symbols
                    /math type:+ num1:2 num2:3
                        options:
                            type: do type:help
                            num1: first number
                            num2: second number

    convert         Converts units into other units
                    /convert type:k>c number:273.15
                    options: 
                        type: what units (kelvin to celcius = k>c)
                        number: the number to convert
        
    remind          Sets a reminder
                    /remind time:8h30m
                        options:
                            time: the time. if there is no unit it defaults to ms
                            reminder: the text to respond with

osu!

 
    accuracycalculator  
                    Returns accuracy and rank based on given values
                    /accuracycalculator 300s:211 100s:12 50s:0 0s:12
                        options:
                            300maxes: mania only. gekis/rainbow 300s/maxes
                            300s: hit 300s (100%)
                            200s: mania only. hit 200s (66.66%)
                            100s: hit 100s (33.33%)
                            50s: hit 50s (16.66%)
                            0s: misses (0%)
                            mode: what mode to calculate for. default osu
                            id: map id for CTB. This option is no longer used

    leaderboard     Returns the top 5 plays on a map
                    /leaderboard id:3217235
                        options:
                            type: global/mods/friends (default global). mods and friend rankings will be added when it's added to API
                            input: mods if type is set to mods, username if type is set to friends
                            id: map id
                            offset: page offset (default 0). 1=pg2, 2=pg3 etc.

    map             Returns information for the map
                    /map id:3217235
                        options:
                            id: map id. default last map requested
                            mods: mods to apply to difficulty calculations. default NM (none)

    osu             Returns information for the user
                    /osu user:SaberStrike
                        options:
                            user: username or ID of player. defaults to the name/id stored in the database
    
    osubest         Returns information for the top 5 plays of all time.
                    /osubest
    
    osubestrs       Returns information for the top 10 plays over the past 24h.
                    /osubestrs
        
    osutop          Returns the top 5 osu! plays for the user
                    /osutop user:15222484
                        options:
                            user: username or ID of player. defaults to the name/id stored in the database
                            offset: page offset. default 0
                            mode: what gamemode. default osu! standard    
                            sort: what to sort by. default pp. score/acc/time
                            
    pp              Returns the pp values for a map
                    /pp id:3217235
                        options:
                            id: id to use. default last map requested
                            mods: mods

    rs              Returns the most recent osu! play for the user. 
                    /rs user:15222484
                        options:
                            user: username or ID of player. defaults to the name/id stored in the database
                            offset: page offset. default 0
                            mode: what gamemode. default osu! standard
                             
    rsplus          Returns the five most recent plays
                    /rsplus mode:mania 
                        options: 
                        user: username or ID of player. defaults to the name/id stored in the database
                        offset: page offset
                        mode: mode

    scores:         Returns all scores on a map for the user
                    /scores sort:Accuracy
                        options:
                            username: username or ID of player. defaults to the name/id stored in the database
                            id: map id. default last stored map
                            sort: default time. score/acc/pp

    setuser:        sets username/mode for the database 
                    /setuser username:mania type:mode
                        options: 
                            username: username/mode
                            type: whether to set mode or username

    skin            Gets the link to my skin(s)
                    /skin
                        options:
                            skin: what skin. default none (sends main page link)

    tsfm:           see scores

    botstatus       Sets the status of the bot.
                    /botstatus type:1 text:osu!

                    1 = playing
                    2 = streaming
                    3 = listening

                            text: text
